На головну

    Підключення парселу:
  • 1) Заходимо в папку де знаходиться parcel-team-project
  • 2) Копіюємо усі файли (крім папки .git)
  • 3) Вставляємо зкопійовані файли у папу проекту
  • 4) Звіряємо версію Node.js за допомогою команди "node -v" (має бути 14+)
  • 5) Встановлюємо всі залежності проекту командою "npm install"
  • 6) Перезаливаємо папку src
  • 7) Заходимо у папку package.json та змінюємо поля "homepage" та "build"
    • "https://goitacademy.github.io/parcel-project-template" змінюємо на "https://OlexandrTkachuk.github.io/goit-js-hw-09 "
    • "parcel build src/*.html --public-url /parcel-project-template/" змінємо на "parcel build src/*.html --public-url /goit-js-hw-09/"
  • 8) Для початку роботи - команда "npm start"
  • 9) Для зупинення роботи - команда "Ctrl + C"
  • 10) Для того щоб почистити кеші проекту - видаляємо папку "parcel-cache"
  • 11) Для того щоб запушити зміни у гітхаб виконуємо наступне:
    • git add .
    • git status
    • git commit -m "..."
    • git push
  • 12) git --> setting --> pages --> branch: "gh-pages" --> save
