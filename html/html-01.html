Основи JavaScript

Структура коду

Інструкції – це синтаксичні конструкції та команди, які виконують дії.
Змінна - це “іменована частинка сховища”, в якій зберігаються дані.
Функції — це головні “будівельні блоки” програми. Вони дозволяють робити однакові дії багато разів без повторення коду.
У JavaScript є вісім основних типів даних.

    Типи даних:
  • Число (Number) +-253 -1
  • BigInt
  • Рядок (String)
  • Булеве значення (Boolean)
  • Null
  • Undefined
  • Object
  • Symbol

alert - показуває повідомлення та чекає, доки користувач не натисне кнопку "ОК";
prompt - показує модальне вікно з текстовим повідомленням, полем, куди користувач може ввести текст, та кнопками ОК/Скасувати
confirm - показуває модальне вікно з питанням question та двома кнопками Ок/Скасувати. Повертає значення true або false

Оператор АБО || - повертає перше правдиве значення або останє, якщо правдивого значення не знайдено;
Оператор І && - повертає перше хибне значення, або останнє значення, якщо хибних значень не було знайдено;
Оператор ?? нульово поєднання - повертає перший аргумент, якшо він не null/undefined. Інакше поверне другий аргумент.

Оголошення Функції (Function Declaration) function ..()

Функціональним Виразом (Function Expression) const ... = function ()

    Методи:
  • String(value) - перетворює значення в рядок
  • Number(value) - перетворює значення в число або NaN
  • Boolean(value) - перетворення булеве
  • (+value) - унарний опертатор, який приводить значення в число або NaN
  • ParseInt(value) - парсить рядок та повертає з нього ціле число
  • ParseFloat(value) - парсить рядок та повертає з нього число з плаваючою комою
  • Math.random() - повертає випадкове число від 0 до 1 (не включаючи 1)
  • Math.max(a,b,c) - повертає найбільше число з довільної кількості аргументів/li>
  • Math.min(a,b,c) - повертає найменше число з довільної кількості аргуентів
  • Math.pow(n, power) - повертає n зведене у ступінь power
  • Math.floor(value) - округлює до нижчого цілого числа
  • Math.ceil(value) - округлює до вищого цілого числа
  • Math.round(value) - округлює згідно правил математики
  • value.toFixed(n) - n вказує скільки цифр залишити в дробі після крапки
  • str.toLowerCase() - змінює регістр усіх символів до малих
  • str.toUpperCase() - змінює регістр усіх символів до великих
  • str.indexOf('substr') - перевіряє наявність елем. substr у рядку str. Якшо існує поверне його індекс, в іншому випадку -1
  • str.lastIndexOf(substr) шукає збіги в зворотньому порядку
  • str.includes(substr) - повертає true/false в залежності від того чи substr є у рядку str
  • str.startsWith(substr) - перевіряє чи починаєтсья str певним substr
  • str.endsWith(substr) - перевіряє чи закінчується str певним substr
  • str.slice(start, end) - повертає частину рядка, починаючи від start і до (але не включно end)
  • arr.pop() - видаляє останій елемент масиву та повертає його
  • arr.push(el...) - додає один або більше елементів в кінець масиву та повертає оновлений масив
  • arr.shift() - видаляє перший елемент масиву та повертає оновлений масив
  • arr.unshift(el...) - додає один або більше елементів на початок масиву та повертає оновлений масив
  • arr.splice(start, deleteCount, el_1,... el_n) - змінує arr, починючи з позиції start: видаляє deleteCount елементів і вставляє el_1 ... el_n елементи
  • arr.concat(arr1, arr2,...) - створює новий масив, в який копіює дані з інших масивів
  • arr.forEach(item, index, array) - дозволяє запустити функцію для кожного елемента масиву
  • arr.find(item, index, array) - перебирає масив поелементно та шукає потрбіний нам елемент. Якщо ф-ція повертає true, пошук припиняється, поверне item.
  • arr.filter(item, index, array) - повертає масив усіх знайдених елементів, або порожній масив якщо нічого не знайшли
  • arr.map(item, index, array) - викликає КБФ на кожному елементі та повертає масив результаті виконання цієї функції
  • arr.sort( (a, b) => a - b) - сортує масив чисел у порядку зростання
  • arr.sort( (a, b) => b - a) - сортує масив чисел у порядку спадання
  • arr.sort( (a, b) => a.localeCompare(b)) - сортує масив рядків у порядку зростання
  • arr.sort( (a, b) => b.localeCompare(a)) - сортує масив рядків у порядку спадання
  • arr.reverse() - змінює порядок елементів масиву на зворотній
  • arr.reduce(acc, item, index, array {}, [initial]) - функція застосовується для кожного елемента масиву по черзі і "переносить" всій результат на настурний рядок. Акумулювання...
  • arr.reduceRight() - аналогічний метод, яки проходить по масиву зправа наліво
  • Array.isArray(value) - повертає true, якщо value - масив, інакше false
  • arr.some() - перевіряє чи хоча б один елемент масиву виконує КБФ
  • arr.every() - перевіряє чи усі елементи масиву виконують КБФ
  • Object.keys(obj) - повертає масив ключів об'єкта
  • Object.values(obj) - повертає масив значень об'єкта
  • Object.entries(obj) - повертає масив пар [ключ, значення]
  • Date().getFullYear() - повертає рік (4 цифри)
  • Date().getMonth() - повертає місяць, від 0 до 11
  • Date().getDate() - повертає день місяця від 1 до 31
  • Date().getDay() - повертає день тижня, від 0 (неділя) до 6 (субота)
  • Date().getTime() - повертає timestamp для дати - кількість ms, що пройшли від 1 січня 1970 UCT +0
  • Date().getTimezoneOffset() - повертає різницю між часовим поясом, у хвилинах
  • Date.now() - без використання NEW повертає поточний timestamp, який не створює проміжного об'єкта Date. Працює швидше і не навантажує "збирач сміття"
  • Date().parse(str) - читає дату з рядка, де формат повинен бути YYYY-MM-DDTHH:mm:ss.sssZ
  • JSON.stringify(value) - перетворює об'єктів у формат JSON
  • JSON.parse(value) - перетворює json.text назад у об'єкт
  • [[Prototype]] - головне сховище методів обєкта
  • obj.hasOwnProperty(prop) - перевіряє чи має обєкт obj властивість prop
